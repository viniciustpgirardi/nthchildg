nth-child Gabarito

A nth-child pseudo-class é especificada com apenas um argumento que descreve o padrão para combinar os indices do elemtno em uma lista de irmãos. O índice do elemento é 1-based. Consulte a documentação sobre nth-child: MDN Web Docs: nth:child()
